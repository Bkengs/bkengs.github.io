js的原生常用操作方法

Array

注：干净表示不改变原数据，脏表示改变原数据

  • concat() : 连接两个或更多的数组，并返回结果。（拼接后的数组）（干净）
  • join() : 通过指定的字符串把数组的内容连接起来。（连接后的字符串）（干净）
  • pop() : 删除并返回数组的最后一个元素。（删除的元素） (脏)
  • push()：向数组的末尾添加一个或更多元素，并返回新的长度。（新数组的长度）（脏）
  • shift()：删除并返回数组的第一个元素 （删除的元素）（脏）
  • unshift()：向数组的开头添加一个或更多元素，并返回新的长度。(新的长度)（脏）
  • reverse()：颠倒数组中元素的顺序。 （颠倒后的数组）（脏）
  • slice()：[+-start,+-end?) 从某个已有的数组返回选定的元素 (选定的数组)（干净）
  • splice()：删除元素，并向数组添加新元素。(删除项数组)（脏）
  • sort(): 对数组进行排序 （排序后的数组）（脏）
  • toString(): 将数组字符串化与join(‘,’)效果相同（返回字符串）（干净）
  • valueOf(): 返回数组的原始对象，其他类型类似（干净）

Number

  • toFixed(): 把 Number 四舍五入为指定小数位数的数字。（干净）

String

  • charAt(): 返回指定位置的字符，超出范围返回空字符串 (干净)
  • concat(): 拼接两个字符串，通常使用’+’来连接（干净）
  • indexOf(): (searchvalue,fromindex?)大小写敏感，返回某个指定的字符串值在字符串中首次出现的位置，找不到返回-1
  • lastIndexOf(): 返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索
  • match()：下一篇博客详细说明
  • replace()：下一篇博客详细说明
  • search()：下一篇博客详细说明
  • slice(): [+-start,+-end?) 从某个已有的字符串返回选定的元素 (选定的字符串)（干净）
  • split(): (separator,howmany?) 把一个字符串分割成字符串数组。可匹配正则(干净)
  • substr(): (+-start, length?)在字符串中抽取从 start 下标开始的指定数目的字符。(新的字符串)（干净）
  • substring(): [+start,+end？)提取字符串中介于两个指定下标之间的字符。(干净)
  • toLowerCase() toUpperCase() (干净)

Function

  • Number(): 把对象的值转换为数字。
  • parseFloat(): 解析一个字符串并返回一个浮点数。
  • parseInt(): 解析一个字符串并返回一个整数。

js中的循环遍历

while,do-while,for(let i, length = arr.length; i < length; i++)这里就不提了

forEach

forEach是数组对象自带的

for…in

for…in主要是用来遍历数组或者对象的键值名，主要用于对象
数组的键名是数字，但是for…in循环是以字符串作为键名“0”、“1”、“2”等等。
for…in循环不仅遍历数字键名，还会遍历手动添加的其他键，甚至包括原型链上的键。
某些情况下，for…in循环会以任意顺序遍历键名。

for…of

for…of是es6中加入的，用来遍历部署了Symbol.iterator属性的数据结构
数组默认部署了Symbol.iterator属性
使用for…of来遍历对象比较麻烦，不推荐。
